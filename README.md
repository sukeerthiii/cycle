# Cycle

Phase-aware fitness log for women. Tracks your menstrual cycle, suggests workouts based on where you are hormonally, and logs everything with full detail. Built as a personal tool — opinionated about body recomposition, rest as training, and training in rhythm with your body.

React + TypeScript + Vite. PWA. All data local (IndexedDB via Dexie.js). No backend.

## Run locally

```
npm install
npm run dev
```
